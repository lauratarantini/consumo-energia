# ⚡ Calculadora de Consumo Elétrico

## 📌 Sobre o projeto
Trata-se de uma calculadora de consumo elétrico inteligenteo, desenvolvida em Python para praticar o ciclo **Entrada → Processamento →Saída**.
O programa solicita o algumas informações básicas sobre o aparelho, como nome e a potência, além do consumo médio por dia, a partir disso é calculado um consumo estimnado por mês.

## Para o calculo, a fórmula utilizada foi **consumo_mensal = (potencia_aparelho * tempo_uso * 30) / 1000**
